เมื่อได้รับพื้นที่ของรูปทรงจะปริ้นเเสดงผลเเละเริ่มเเสดงกราฟรูปทรง

  input [(1, 1), (4, 2), (3, 5), (1, 4)]
  = 1/2 * |(1×2 + 4×5 + 3×4 + 1×1) - (1×4 + 2×3 + 5×1 + 4×1)|
  = 1/2 * |(2 + 20 + 12 + 1) - (4 + 6 + 5 + 4)|
  = 1/2 * |35 - 19| = 1/2 * 16 = 8
   >>> พื้นที่จากพิกัดจุด = 8.0 ตารางหน่วย
